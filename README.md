## User Guide ##
代码在windows上开发，环境为IPy-0.83+gevent-1.0.1+python-2.7.7
如果在python-2.7.8及以下版本中使用请先安装pip然后

`pip install gevent`

`pip install IPy`

如果是python-2.7.9+就可以直接安装gevent和IPy啦

> 用pip的原因是windows下面pip能愉快的装上gevent╮(￣▽￣")╭ 

然后进入命令行，`python Main_Gevent.py` 就好啦~~~
